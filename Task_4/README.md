# Домашнее задание 4: Визуализация структуры белка

## Описание задания

Задание включает выбор и установку программного обеспечения для визуализации белковых структур, загрузку структуры белка из Protein Data Bank, изучение возможностей ПО и создание различных типов визуализации с различными способами раскраски.

## Выбранное программное обеспечение

**RasMol**  
Официальный сайт: http://www.openrasmol.org/

RasMol — это программа для визуализации молекулярных структур, разработанная Роджером Сейлором. Программа позволяет отображать структуры белков, нуклеиновых кислот и других молекул в различных режимах визуализации.

## Выбранная структура белка

**PDB ID:** 1TIM
**Ссылка:** https://www.rcsb.org/structure/1TIM 
**Название белка:** Триозофосфатизомераза (Triosephosphate isomerase, TIM)
**Описание:** Триозофосфатизомераза — это димерный фермент гликолиза, катализирующий взаимопревращение дигидроксиацетонфосфата и D-глицеральдегид-3-фосфата. Структура 1TIM представляет собой структуру триозофосфатизомеразы из курицы (Gallus gallus).

## Выполненные этапы

### 1. Установка программного обеспечения

- Зарегистрированы и получены необходимые лицензии (бесплатные)
- Скачана и установлена программа RasMol с официального сайта
- Изучены инструкции и туториалы по использованию программы

### 2. Загрузка структуры белка

- Выбрана структура белка среднего размера на Protein Data Bank (https://www.rcsb.org/)
- Структура скачана в формате PDB (одна из сборок белка)
- Структура загружена в RasMol

### 3. Визуализация структуры

Выполнены следующие типы визуализации:

#### a. Wireframe (Проволочная модель)
- **Файл:** `1.jpg`
- **Описание:** Отображение структуры в виде проволочной модели, где атомы представлены точками, а связи — линиями

#### b. Backbone (Остов)
- **Файл:** `2.jpg`
- **Описание:** Визуализация основного остова белковой цепи (α-углеродные атомы)

#### c. Spacefill (Заполнение пространства)
- **Файл:** `3.jpg`
- **Описание:** Отображение атомов в виде сфер с ван-дер-ваальсовыми радиусами

#### d. Ribbons (Ленты)
- **Файл:** `4.jpg`
- **Описание:** Визуализация в виде лент, показывающая вторичную структуру (α-спирали, β-листы)

#### e. Molecular surface (Молекулярная поверхность)
- **Файл:** `5.jpg`
- **Описание:** Отображение молекулярной поверхности белка

### 4. Раскраска структуры

#### a. Цветовая модель CPK
- **Файл:** `6.jpg`
- **Описание:** Раскраска по модели CPK (Corey-Pauling-Koltun), где каждый химический элемент имеет свой характерный цвет

#### b. Различные цвета по доменам (частям) белка
- **Файл:** `high_quality.jpg`
- **Описание:** Раскраска различных доменов или частей белка разными цветами для лучшей визуализации структурных элементов
