# Домашнее задание 5: Предсказание и сравнение третичных структур белков

## Описание задания

Задание включает предсказание третичной структуры белка с помощью двух различных инструментов фолдинга, парное выравнивание полученных структур и их визуализацию с анализом совпадения предсказаний.

## Исходные данные

### Первичная последовательность белка

```
MKGMLTGPVTILNWSWPREDITHEEQTKQLALAIRDEVLDLEAAGIKIIQIDEAALREKLPLRKSDWHAKYLDWAIPAFRLVHSAVKPTTQIHTHMCYSE
```

**Длина последовательности:** 100 аминокислотных остатков  

### Инструменты фолдинга белков

1. **ESMFold** - инструмент на основе языковой модели ESM (Facebook Research)
2. **AlphaFold2** - инструмент от DeepMind (через ColabFold)

### Инструмент парного выравнивания

**jFATCAT-rigid** (в списке Alignment Method)  
URL: https://www.rcsb.org/alignment

## Выполненные этапы

### 1. Предсказание структур

#### ESMFold
- **Ноутбук:** `ESMFold.ipynb`
- **Результат:** `ptm0.747_r3_default.pdb`
- **Параметры качества:**
  - PTM (Predicted TM-score): 0.747
  - pLDDT (predicted LDDT): 78.835
  - Количество рециклов: 3

#### AlphaFold2
- **Ноутбук:** `AlphaFold2.ipynb`
- **Результат:** `HW5_ESMFold_00cf3_unrelaxed_rank_001_alphafold2_ptm_model_5_seed_000.pdb`
- **Параметры качества (лучшая модель):**
  - pLDDT: 93.6-94.8
  - pTM: 0.828-0.849
  - Количество рециклов: 3
  - Модель: alphafold2_ptm_model_5

### 2. Парное выравнивание структур

Выравнивание выполнено с помощью инструмента **jFATCAT-rigid** на сайте RCSB PDB Alignment:
- Использованы Assembly 1, Chain A для обеих структур
- Метод выравнивания: jFATCAT-rigid
- Результаты выравнивания визуализированы в PyMOL (см. раздел "Визуализация")
- Выравнивание показало структурное сходство между предсказаниями AlphaFold2 и ESMFold

### 3. Визуализация

Визуализация выполнена в **PyMOL**:
- **Сессия PyMOL:** `pymol_session.pse`
- Выполнена раскраска выравнивания двумя различными цветами по цепям (Chain A, Chain B)
- Созданы скриншоты визуализации:
  - `photo_2025-11-21_22-03-45.jpg`
  - `photo_2025-11-21_22-18-39.jpg`

## Использованные инструменты и ресурсы

1. **Google Colab** - платформа для выполнения ноутбуков
2. **ColabFold** - реализация AlphaFold2 для Google Colab
3. **ESMFold** - инструмент предсказания структуры на основе языковой модели
4. **RCSB PDB Alignment** - веб-инструмент для выравнивания структур

## Примечания

- Обе структуры предсказаны для одной и той же последовательности длиной 100 аминокислотных остатков
- AlphaFold2 показал более высокие показатели уверенности (pLDDT > 93) по сравнению с ESMFold (pLDDT ≈ 79)
- Обе структуры содержат одну цепь (Chain A)
